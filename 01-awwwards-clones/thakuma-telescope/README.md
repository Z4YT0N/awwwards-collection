# Telescope — A Curated Visual Experience

This project is a React-powered recreation of the stunning scroll and gallery experiences found in modern web design. 

## Inspiration
I took inspiration from the original website [telescope.fyi](https://telescope.fyi/). This project aims to replicate the surreal frames, layered transitions, and smooth motion that make the original site so iconic.

## Key Features
- **Dynamic Projects Banner**: Layered image reveal animations using mask-image and GSAP.
- **Vertical Title Reveals**: Staggered character-level animations that react to scroll position.
- **Arc-Point Gallery**: A unique gallery system where thumbnails follow a Bezier path, paired with background swapping and focused title states.
- **Smooth Locomotion**: Powered by Lenis for high-end momentum scrolling.

## Upcoming Changes
This project is an evolving exploration. Some planned updates include:
- **New Creative Sections**: Adding even more unique ways to display curated content.
- **UI & Aesthetic Polish**: Ongoing refinements to micro-animations and layout transitions.
- **Performance Optimizations**: Further smoothing of WebGL-like effects in a standard DOM environment.

## Built With
- **React**: Component-driven architecture.
- **GSAP (GreenSock)**: Industry-standard animation engine.
- **Lenis**: Lightweight smooth scrolling.
- **Vite**: Ultra-fast build tool and dev server.
