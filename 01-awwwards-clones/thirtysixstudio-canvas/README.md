## 🚀 Stunning ThirtyStudio-Inspired Landing Page with Advanced GSAP Animations

A pixel-perfect recreation of the visually striking ThirtySixStudio website, featuring sophisticated animations, interactive elements, and fluid user experience. This project showcases modern web animation techniques using GSAP and React to create an immersive digital experience.

## ✨ Features

- **Mesmerizing Animations**: Smooth, frame-by-frame animations using GSAP and canvas elements
- **Interactive UI Elements**: Engaging click interactions with animated transitions
- **Diagonal Text Reveals**: Text elements that reveal with an elegant diagonal animation on scroll
- **Responsive Design**: Fully responsive layout built with Tailwind CSS
- **Smooth Scrolling**: Integrated Locomotive Scroll for butter-smooth scrolling experience
- **Modern React Architecture**: Built with React 19 and Vite for optimal performance

## 🛠️ Technologies Used

- **React**: Core framework for building the UI components
- **GSAP (GreenSock Animation Platform)**: Powers the advanced animations
- **Locomotive Scroll**: Enables smooth scrolling functionality
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Vite**: Fast and efficient build tool
- **Canvas API**: Used for rendering dynamic animations

## 📋 Project Structure

The project is organized into key components:

- **App.jsx**: Main application component that handles the overall layout and structure
- **Canvas.jsx**: Handles the dynamic canvas animations that appear throughout the page
- **DiagonalReveal.jsx**: Creates the eye-catching diagonal text reveal animations
- **canvasImages.js**: Stores references to all animation frames for the canvas elements
- **data.js**: Contains configuration data for positioning and timing of animations

## 🏃‍♀️ Getting Started

### Prerequisites

- Node.js (v16+)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/pyna-b/React-GSAP-Animated-Studio-website.git
   cd React-GSAP-Animated-Studio-website
   ```

2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```

3. Start the development server:
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. Open http://localhost:5173 to view it in your browser.

## 🎨 Animation Details

This project implements several advanced animation techniques:

- **Frame-by-frame Animation**: Using sequences of images for smooth pepper animations
- **Growing Circle Animation**: Interactive growing circle effect when clicking on the studio name
- **Scroll-triggered Animations**: Elements that animate as they enter the viewport
- **Text Reveal Effects**: Text that reveals with stylish diagonal transitions
- **Mouse-follow Elements**: UI elements that follow cursor movement

## 🔍 How It Works

The animations are powered by GSAP and integrated with React's component lifecycle. The Canvas component handles frame-by-frame animations using HTML5 Canvas, while the DiagonalReveal component creates text reveal effects as the user scrolls through the page.

Locomotive Scroll enhances the scrolling experience, and Tailwind CSS provides responsive styling throughout the application.

## 🌐 Inspiration

This project is a recreation of ThirtySixStudio's innovative website design, showcasing similar animation techniques and visual aesthetics. It serves as both a tribute to their excellent design work and a demonstration of implementing complex animations in React.
